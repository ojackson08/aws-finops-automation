# Security Policy

## About This Project

`aws-finops-automation` is an **infrastructure hygiene tool** developed by Merkaba AI Risk Management.

## Reporting a Vulnerability

If you discover a security vulnerability — including logic flaws that could cause the deletion of active, attached resources — please report it to:

**Email:** security@merkabacreatives.org
**Subject line:** `[SECURITY] aws-finops-automation — <brief description>`

We will acknowledge receipt within 48 hours.
